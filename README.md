# immersionToolbox

The ImmersionToolbox is a critical component required to realize the vision of the room as an integrated instrument and is under active development at the MIT.nano Immersion Lab. It consists of a set of tools for the acquisition, analysis, visualization, and management of multimodal data acquired at the Immersion Lab.

The toolbox currently interfaces with motion tracking data acquired through the optitrack system, physiological recordings acquired through the Delsys system, audio and accelerometer data acquired through National Instruments DAQ systems, ultrasound imaging data acquired using the Cephasonics ultrasound system. It also provides a script for automating photogrammetry data processing by interfacing with the reality capture software.

The Immersion Toolbox provides an interface to blender software for visualizing motion capture data in 3D. Furthermore, it has python scripts to ease the burden of data management and synchronization acquired using multiple modalities.
